# Customer Service

It's the most simple multi-agent example.

## How to set up

```bash
# Create your .env file (use the .env.example as a template)
cp .env.example .env

# Install dependencies
uv sync
```

## How to run locally

```bash
# Activate the virtual environment
source .venv/bin/activate

# Windows
.venv\Scripts\activate

# Run the agent
adk web
```
